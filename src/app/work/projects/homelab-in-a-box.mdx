## What it is

Homelab in a Box is a “start here” self-hosting project: a small set of useful services, deployed with Docker Compose, behind a reverse proxy, with sane security defaults.

## Focus

- Repeatable setup steps (bootstrap → deploy → backup → restore)
- Minimal magic; understandable configs
- Secure remote access patterns (Cloudflare + tunnels where appropriate)
