## What it is

TabStack is a local-first tool for organizing browser sessions around projects: save a set of tabs with context, keep them grouped, and bring them back when you need them.

## Notes

- Chrome extension
- Productivity / workflow focus
- Built for repeatable, “pick it back up later” work
